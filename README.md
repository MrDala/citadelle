# Citadelle

## Présentation

Reproduction du jeu de société Citadelle.
Règles du jeu suivies : [Règles 2ème édition](https://www.jeuxavolonte.asso.fr/regles/citadelles_seconde_edition.pdf)

## Serveur
Serveur développé en Node.js

Script de démarrage.

```zsh
cd server
npm install
npm start
```
